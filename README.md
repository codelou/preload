# 前言

此项目是本人开始学习jquery插件写法的试水项目，但是很有借鉴作用，主要值得学习的地方是jquery插件学习思想以及如何
搭配gulp去压缩打包一个js文件为min.js文件

## 技术栈
jquery + gulp

## 项目运行
git clone https://github.com/codelou/preload.git

cd preload

npm install 

npm run build  (输出压缩打包后的preload代码)
